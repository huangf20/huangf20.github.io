Git常用命令

git

获取Git配置信息，执行以下命令：

git config --list

仓库状态

git status

版本回退

git log

在 Git中，用HEAD表示当前版本，也就是最新的提交commit id，上一个版本就是HEAD^ ，上上一个版本就是HEAD^^，当然往上100个版本写100个^比较容易数不过来，所以写成HEAD~100。
现在我们要把当前版本回退到上一个版本，就可以使用git reset命令：

git reset --hard HEAD^

Git提供了一个命令git reflog用来记录你的每一次命令：

git reflog

创建与合并分支

首先我们创建dev分支，然后切换到dev分支：

git checkout -b dev

git checkout命令加上-b参数表示创建并切换，相当于以下两条命令：

git branch dev
git checkout dev

然后用git branch命令查看当前分支

git branch

我们在dev分支上进行添加修改操作，然后我们把dev分支的工作成果合并到master分支上：

git checkout master
git merge dev

git merge命令用于合并指定分支到当前分支

删除dev分支：

git branch -d dev

如果要丢弃一个没有被合并过的分支，可以通过git branch -D 强行删除。

配置别名

我们只需要敲一行命令，告诉Git，以后st就表示status：

git config --global alias.st status

合并分支并线性提交历史 

git checkout bugFix 
git rebase master

撤销更变：

1.git reset:

2.git revert(远程分享时只能用此方法)

提交更变到当前的位置

git cherry-pick C2 C4

------>

交互式更变提交：
git rebase -i HEAD~4

------>

从远程仓库获取数据：
git fetch
从远程仓库获取数据并更新本地分支：
git pull origin master
